# Portfolio of Rutwij - Personal Website

This repository contains the source code for my personal portfolio website, a digital reflection of my personal brand, values, and journey.

Built with Next.js, Tailwind CSS, and Framer Motion, this site is designed to be a fast, responsive, and visually engaging showcase of my work, career, and interests.

***

### Key Features
- **Scroll-based Introduction:** An engaging hero section with parallax effects.
- **Section Previews:** Snippets of each main section on the homepage.
- **Static & Fast:** Built with Next.js for great performance and SEO.
- **Dark Theme:** A clean, black and white theme with vibrant accents.
- **Responsive:** Looks great on desktop, tablet, and mobile.

### Tech Stack
- **Frontend:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Deployment:** [Vercel](https://vercel.com/)

***

### Running Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rutwiz/portfolio-website.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd portfolio-website/app
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
The application will be available at `http://localhost:3000`.
